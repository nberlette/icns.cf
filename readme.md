## Schema

<pre>
1.  icns.ml / <strong>slug*</strong> <em>‚Äî color</em> . <em><strong>type</strong></em>‚Ä†

2.  icns.ml / <em>color</em> / <strong>slug*</strong> . <em><strong>type</strong></em>‚Ä†
</pre>

|  **Param**         | **Usage**    | **Syntax Info**                                             | **Examples**               |
| -----------------: | :----------- | :---------------------------------------------------------- | :------------------------- |
| <kbd>**`slug`***</kbd> | **required** | Alphanumeric. **See: [naming conventions]**                 | `nextdotjs`, `500px`       |
| <kbd>`color`</kbd> | **optional** | CSS valid: **hex** (3/4/6/8), **name**, **rgb**, ...        | `fff`, `red`, `rgb(0,0,0)` |
| <kbd>**`type`**‚Ä†</kbd> | **advised**  | **`.svg`** or **`.png`** *(raster support coming soon)*     | `.svg`, `.png`             |
